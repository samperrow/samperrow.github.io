Preconnect

Used to resolve the DNS, TCP, and SSL handshake before the main resource is downloaded.

Enter only the domain name of the resource!

Preconnects can be utilized via HTML elements or in the HTTP header:

HTML: <link rel=”preconnect” href=”https://ajax.googleapis.com”>

HTTP Header: “Link: <https://fonts.googleapis.com/> rel=preconnect”

*Preconnect should only be used for the most critical resources. DNS prefetch should be used for the others.

*Some resources such as fonts are loaded in anonymous mode. In such cases you should set the crossorigin attribute with the preconnect hint. If you omit it, the browser will only perform the DNS lookup.

Prefetch

Used to load resources before they are needed, which are cached.

Resources should be on the same site, not from external sources like CDN’s, and low priority!

A common use case is to have a simple website landing page that fetches heavier resources used by the rest of the site.


HTML elements or HTTP header can be used:

<link rel=”prefetch” href=”https://example.com/critical-js-script.js”>

<link rel=”prefetch” href=”https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js”>

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/rel/prefetch

DNS Prefetch

This resolves the DNS of an external resource (“cross origin”) ahead of time. Don’t use this to load resources on the current site.


Enter only the domain name of the resource:

HTML example: <link rel=”dns-prefetch” href=”https://ajax.googleapis.com”>

HTTP Header: “Link: <https://fonts.googleapis.com/>; rel=dns-prefetch”


The above HTML can be used if the JS script below were loaded.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
